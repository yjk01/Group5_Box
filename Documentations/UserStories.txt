What the user wants out of the project:

- A box that is able to hold things
- Easy assembly for the customer
- Has no rough edges
- Box has tabs to hold sides together
- User can run on command prompt
- User can put in inputs of size x,y,z which correspond to length, height, and width of box 
- User can decide which material they are using
- User can decide whether they want a normal box or a phone holder
- Software will take whatever lengths were given by user and convert the size of box to whatever user wants
- Box (Phone holder) could hold the phone and have it charging at the same time
