Instructions for Phone Box assembly:
1) Insert two tabs into inserts on the front face
2) Build box: side with inserts and holes is front, 
   side with hole is top, bottom is same size as top, 
   back is same size as front, two sides.
3) Assembly is done

Instructions for Phone Box use:
1) Insert phone into hole on top of box
2) Tilt phone until it is visible through front hole
3) Insert charger into phone
4) Set phone on tabs to let it rest, charger should sit in the indentation
5) Plug charger into wall, voala!
